# TO-DO LIST RPG

> 할 일을 기록하면 RPG 퀘스트 형태로 변환되고, 퀘스트를 완료하여 경험치를 쌓아 레벨업도 하고, 희귀 아이템도 얻고, 업적도 쌓고

## 기능

### 1. 할 일 등록

- 일반적인 To do list 들과 동일한 UI/기능으로 작성
- 할 일, 마감일을 입력
- 할 일을 삭제하면 완전 삭제하지 않고 취소 표시로 남겨둠

### 2. 퀘스트 생성

- 입력한 할 일을 분석해서 AI로 퀘스트를 생성
- 생성할 RPG 의 배경은 위쳐/스카이림/발더스 게이트/D&D 등의 서양 판타지 세계관
- 퀘스트 생성시 난이도, 경험치를 계산해서 반환

### 3. 퀘스트 완료

- 퀘스트 완료시 경험치 획득
- 마감일을 지나면 날짜가 지날수록 경험치 감소 (획득 최소치 1)

## 이후 추가할 기능

### 1. 아이템 획득

- 퀘스트 완료 시 일정 확률로 아이템 획득
- 아이템 등급에 따라 다른 확률로 획득
- 획득한 아이템은 [가방] 메뉴에서 확인 가능
- 아이템은 메인 페이지에 자유롭게 배치 가능 (드래그 앤 드롭으로 위치 설정)

### 2. 업적 획득

- 특정 행동 완수하면 업적과 칭호 획득 가능
- 획득 가능한 업적과 획득한 업적, 칭호는 [업적] 메뉴에서 확인 가능
- 획득한 칭호는 메인 페이지에 자유롭게 배치 가능 (드래그 앤 드롭으로 위치 설정)
